What is Markdown?

Markdown is a lightweight markup language designed to simplify HTML writing. It uses a plain text format that is both easy to read and write, and can be converted into various formats, including HTML. Due to its simplicity and user-friendliness, Markdown is widely used for document writing, blog posts, emails, and more.

Headings

In Markdown, headings are represented by the hash symbol (#), with the number of hash symbols indicating the heading level, up to six levels.

# Heading 1
## Heading 2
### Heading 3

Lists

Markdown supports both ordered and unordered lists. Unordered lists can be created using asterisks, plus signs, or hyphens, while ordered lists use numbers followed by a period.

- Unordered list item 1
- Unordered list item 2

1. Ordered list item 1
2. Ordered list item 2

Blockquotes

Quoted text is indicated by the greater-than symbol (>).

> This is a quoted text.

Links

Links are created using square brackets and parentheses, where the square brackets contain the link text and the parentheses contain the link address.

[OpenAI](https://www.openai.com)

Images

Inserting an image is similar to inserting a link, but with an exclamation mark preceding the square brackets.

![Image Description](Image Link)

Bold and Italics

Text can be made bold by enclosing it with double asterisks and italicized with a single asterisk.

**Bold text**
*Italic text*

Code

Inline code is enclosed in backticks, while code blocks are created using three backticks.

`Inline code`
